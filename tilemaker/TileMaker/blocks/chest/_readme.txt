This is where to put the chest texture PNG files:

    ender.png
    normal.png
    normal_double.png

from the "\assets\minecraft\textures\entity\chest" directory in a Minecraft resource pack in order to make terrainExt.png.

See mineways.html (in the docs directory, or http://mineways.com/mineways.html) about the TileMaker for more information.
